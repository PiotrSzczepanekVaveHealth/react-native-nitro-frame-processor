SDK Contents

Description of the included files and directories:

Files		Explanation
-----------------------------------
include/cvie.h
                This is the header file for the CVIE library.

include/cvie2.h
                Header for the CVIE2 API which must be used for 3D processing.

bin/cvie64.framework
                This is the CVIE library.

bin/us2d-7_di.h
                Definitions for the Doctor's Interface parameters.


images/
                This directory contains one or more test images.

volumes/
                This directory contains one or more test volumes.

par/
                This directory contains one or more default parameter files.
                To use for testing only.
			
READMESDK.txt
                This text file.
